# Student Grade Calculator

## 📌 Overview
A Java console application that calculates a student's total marks, average, grade, and pass/fail status based on marks entered for five subjects.

## ✨ Features

- Student Information Input
- Marks Entry for 5 Subjects
- Total Calculation
- Average Calculation
- Grade Calculation (A–F)
- Pass/Fail Evaluation
- Clean Console Report

## 🛠️ Technologies Used

- Java
- Eclipse IDE

## 📂 Project Structure

```
StudentGradeCalculator
└── src
    └── com.mits.gradecalculator
        └── Main.java
```

## 🚀 How to Run

1. Clone the repository.
2. Open it in Eclipse.
3. Run `Main.java`.
